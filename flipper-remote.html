Pair over Bluetooth

Connects to your Flipper Zero (or any BLE device) using your browser's Web Bluetooth. Turn on Bluetooth on the Flipper: Settings → Bluetooth → On, then tap Pair.

Connect over Wi-Fi

Phones can't join a Wi-Fi network from a web page, so this connects to a device already on your local network by its address (e.g. a Flipper with the Wi-Fi Dev Board, or an ESP-based bridge exposing a WebSocket).

The page is served over HTTPS on GitHub Pages, so browsers block plain ws:// to a local IP. Run this page locally (or use a wss:// endpoint) to talk to a real device. Otherwise it stays in demo mode.
